# Mindwtr 0.6.20

## Highlights
- Added bulk actions in Desktop Contexts view so you can select multiple tasks and apply updates together.
- Fixed intermittent mobile startup issues by guarding keep-awake activation and startup widget scheduling.
- Improved desktop startup polish by preventing terminal/i18n-key flashes.
- Hardened Linux AppImage packaging by replacing broken absolute `.DirIcon` symlinks during CI repair.

## Desktop
- Added multi-select bulk workflow in Contexts view for faster triage/edit operations.
- Reduced startup UI noise and translation-key flicker on launch.

## Mobile
- Guarded keep-awake activation to avoid startup-time runtime failures on some devices.
- Guarded startup widget scheduling paths to avoid timer-related initialization errors.

## CI / Packaging
- Fixed AppImage metadata repair to replace broken absolute `.DirIcon` symlinks with a valid icon file.
